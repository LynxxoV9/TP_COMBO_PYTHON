# TP4 – Manipulation de chaînes de caractères

Ce travail pratique porte sur la manipulation des chaînes de caractères en Python à travers l’analyse d’une phrase saisie par l’utilisateur.

Le programme met en œuvre différentes méthodes intégrées permettant de traiter du texte, d’extraire des informations utiles et d’effectuer des vérifications logiques.

## Objectifs du TP

Ce TP vise à :
- Manipuler des chaînes de caractères
- Découper une phrase en mots
- Effectuer des analyses simples sur du texte
- Appliquer des traitements conditionnels
- Comprendre les méthodes intégrées des chaînes en Python

## Fonctionnalités du programme

Le programme permet de :
- Demander une phrase à l’utilisateur
- Compter le nombre de mots de la phrase
- Identifier le mot le plus long
- Vérifier si la phrase est un palindrome
- Afficher les résultats de manière claire

## Concepts Python utilisés

Notions mises en pratique :
- Chaînes de caractères (str)
- Méthodes des chaînes
- Listes
- Boucles for
- Conditions if / else
- Manipulation de texte

## Exécution du programme

Depuis le dossier TP4 :
```bash
python tp4.py

Le programme s’exécute en mode interactif et affiche les résultats de l’analyse.
